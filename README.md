## HTML - JS - CCS
### Nuestra idea principal fue ver que tan de acuerdo estabamos en cosas generales y cuantas personas pensaban lo mismo, por eso pensamos en el programa de television "100 Argentinos dicen". 
### En nuestro juego queriamos que las preguntas sean de todo tipo pero que tambien sean aleatorias asi no les toca la misma todo el tiempo. Tambien que cada pregunta tenga un limite de tiempo asi no se vuelve tan lento, ya que si se termina el tiempo perdes el punto. 

### Para poder hacer esto primero nos dividimos las partes y los tiempos con diagramas de Gantt, asi poder llegar bien a la entrega. Despues hicimos los HTML uno para cada parte del juego y una para el usuario, en esta pusimos los titulos, input,  botones y enlaces. ###Despues empesamos a programar el js, con esto tambien a buscar informacion para poder ponerle un limite de tiempo a el juego. En el js Hicimos los gets generales, los pedidos espesificos (puntajes, partidas, preguntas y sus opciones, usuarios registrados, admins), los posts, puts, deletes.
### Mientras se hacia esto empesamos a hacer el ccs, intentamos que se paresca lo mas parecido posible a la presentacion previa con cololes celesteas, por elcolor del colegio, un poco de encaje y letra entendible.
